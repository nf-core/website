## RFCs aka Request For Comments

This new initiative coming from the core team is meant to help formalize and present substantial ideas to improve the nf-core.

A discussion can be started on Slack, which can lead to an issue on nf-core/proposals, which in the end can lead into a Pull Request.
For more information, please read [the documentation](https://nf-co.re/docs/contributing/project_proposals).
The important part is that we maintainers will discuss these proposals and coordinate with the core team to help port all of these to fruition.
