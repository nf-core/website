This hackathon project focuses on creating new
[nf-core/modules](https://github.com/nf-core/modules) that we and others might find useful to use.
These modules will directly be implemented in our pipelines which are currently in the process of being
rewritten to use Nextflow / nf-core standards. We use this to make Nextflow and nf-core more popular
at our Institute.

---

## Goal

Design and verify new modules following the nf-core framework and contribute them so they
become available for use by the larger community. These modules should be in need by someone
in the team, so they can directly include them in their new / rewritten pipelines.

---

## Recommended preparation

Participants should ideally have:

- Basic familiarity with Git and GitHub (forking repositories, creating branches, and opening pull requests)
- Basic knowledge of Nextflow
- Familiarity with nf-core modules

The following training material is recommended:

- [Hello nextflow](https://training.nextflow.io/latest/hello_nextflow/)
- [Hello nf-core](https://training.nextflow.io/latest/hello_nf-core/)
