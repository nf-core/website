This project aims to update the [nf-core/chipseq](https://nf-co.re/chipseq/) pipeline to meet requirements of recent
Nextflow versions (>25.10). Additionally we will address open issues and milestones for the release of pipeline version 2.2.0.

## Goal

1. Updating subworkflows and modules to be compatible with nextflow (>25.10)
2. Address open issues, bugs and milestones for the release of version 2.2.0.
3. Implementing new functionalities.

-> Happy to brainstorm further goals.

## Prior training

📚 We encourage participants to have completed the following Nextflow training materials:

- [Hello nextflow](https://training.nextflow.io/latest/hello_nextflow/)
- [Nextflow run](https://training.nextflow.io/latest/nextflow_run/)
