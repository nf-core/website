This project focuses on improving [LRSomatic](https://github.com/intgenomicslab/lrsomatic) pipeline.

### Where

Preferably in person at the local Brussels, Belgium site - but anybody's welcome online as well.

_We welcome contributors of all experience levels._


## Tasks

Participating in this team could include the following tasks:

- Adding longphase-S(TO)
- Using longphase-S(TO) to improve purity/ploidy prediction in Wakhan/ASCAT
- Add more elaborate QC (also whatshap stats)
- Improve documentation
- Port local modules to nf-core
