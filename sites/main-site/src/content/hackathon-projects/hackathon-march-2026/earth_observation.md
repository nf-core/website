This project focusses on a new release for the [nf-core/rangeland pipeline](https://nf-co.re/rangeland/1.0.0/).
The release should primarily replace local modules with nf-modules, and update the pipeline to use the newest version of the [FORCE tool](https://force-eo.readthedocs.io/en/latest/index.html).

## Tasks

- [#27](https://github.com/nf-core/rangeland/issues/27): Improve the pipeline diagram
- [#35](https://github.com/nf-core/rangeland/issues/35): Update the FORCE version, this should also be done for force nf-core modules
- [#37](https://github.com/nf-core/rangeland/issues/37) Port local modules to nf-core
- [#38](https://github.com/nf-core/rangeland/issues/38) Replace local modules with nf-core modules
- Optional: [#36](https://github.com/nf-core/rangeland/issues/36): Improve nf-test tests

## Where

Berlin, online participation is possible.
