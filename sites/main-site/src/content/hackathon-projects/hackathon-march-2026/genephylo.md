This project aims to make progress the first version of `genephylo`.

We welcome contributors at any level of experience.

## Goal

Add new features to the pipeline, develop the corresponding nf-core modules and deliver a first working version of the pipeline.

## Tasks

1. Write module for macse refineAlignment and implement to pipeline
2. Write module for treerecs and implement to pipeline
3. Implement ORF detection
4. Extract and convert relevant sections of multiplesequencealign as a subworkflow
5. Integrate subworkflow into the pipeline

## Where

In-person (Okinawa hub) and online.
