## Goal

Contribute to the [nf-core/genephylomodeler](https://github.com/ninaxiong11/nf-core-genephylomodeler) pipeline and prepare for a minimal first release. Currently, we aim to add support for more analysis tools such as codeml as well as the DRHIP module for generating tabular summaries from HyPhy outputs.

## What participants will do

Each contributor will:

1. Choose a module to work on with the team
2. Add it to the pipeline
3. Run tests and lint checks locally.
4. Open a Pull Request for review.

## Recommended preparation

Participants should ideally have:

- Basic familiarity with Git and GitHub (forking repositories, creating branches, and opening pull requests)
- Basic knowledge of Nextflow
- Familiarity with nf-core modules and pipelines
