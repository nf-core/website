nf-core has several leadership teams that help run the community:

- **[Core team](https://nf-co.re/governance#core-team):** Guides community-wide decisions and handles administrative access
- **[Maintainers team](https://nf-co.re/governance#maintainers):** Oversees modules, pipelines, and test configs
- **[Outreach team](https://nf-co.re/governance#outreach):** Coordinates nf-core events and the bytesize series
- **[Infrastructure team](https://nf-co.re/governance#infrastructure):** Maintains the nf-core template, tooling, GitHub Actions, and AWS
- **[Documentation team](https://nf-co.re/governance#documentation):** Maintains quality and consistency across nf-core documentation
- **[Safety team](https://nf-co.re/governance#safety):** Handles inter-personal conflicts and code of conduct matters

See the [Governance](https://nf-co.re/governance#governance) page for a full description of the roles and responsibilties of these teams.
