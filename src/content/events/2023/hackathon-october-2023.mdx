# Welcome

Join us in-person or online for the October 2023 nf-core hackathon!

This hackathon will be held in advance of the _**Nextflow Summit 2023**_ in Barcelona, Spain üá™üá∏. You can find out more information about the summit at [https://summit.nextflow.io](https://summit.nextflow.io).

In addition to the in-person event, we will run a free virtual option online in [gather.town](https://gather.town/).

:::info
This hackathon is not meant to be a training event.
It is aimed at people that already have a basic understanding of Nextflow and/or nf-core.

If you're interested in learning Nextflow and nf-core, there are free online workshops in September.
For more information and the training registration link, see the

## Hackathon focus: nf-test

At this hackathon, our special topic is [nf-test](https://code.askimed.com/nf-test/) üöÄ

nf-test is a testing framework for Nextflow code.
We want to start using it for all nf-core pipelines and modules.
Some proof-of-concept work has already been done, with standardised implementation
patterns mostly worked out, but there's still a lot of work to add this in for all components.

There are nf-test related projects in several of the groups below.
It would be great to move nf-test adoption forward during the event!
